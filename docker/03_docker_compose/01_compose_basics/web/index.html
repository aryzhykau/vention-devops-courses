Docker Compose Demo

Hello from Docker Compose!

Это демонстрационная страница для задания по основам Docker Compose.

Информация о странице

Эта страница обслуживается Nginx контейнером. Вы можете изменить её содержимое в папке web.

Многоконтейнерное приложение

Наше приложение состоит из двух сервисов:

  • Веб-сервер (nginx) - обслуживает эту страницу
  • База данных (redis) - хранит данные

Полезные команды Docker Compose

  • docker-compose up -d - запустить сервисы в фоновом режиме
  • docker-compose down - остановить и удалить контейнеры
  • docker-compose ps - показать список запущенных контейнеров
  • docker-compose logs - показать логи
